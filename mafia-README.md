# 🍕 Mafia — App de Conteo de Inventario

App web mobile-first para el equipo de cocina de Mafia. Permite hacer conteos de stock rápidos y enviar el resumen al grupo de WhatsApp.

## ✨ Funcionalidades

- Selección de usuario (Blanca, Elizabeth, Eva, Valentina, Yhonathan, Otra)
- Turno AM / PM con detección automática de día
- Conteo por 5 secciones: Proteínas y Embutidos, Quesos, Salsas y Otros, Verduras, Masas e Ingredientes
- Marcado manual de urgencia ⚠️
- Resumen con vista previa del mensaje
- Envío al grupo por WhatsApp
- Panel de administrador con clave

## 🚀 Deploy en Netlify

1. Sube este repositorio a GitHub
2. Entra a [netlify.com](https://netlify.com) y conecta el repo
3. Publish directory: `.`
4. ¡Listo!
